## Overview

Writes the list of files in the Vungle cache to the log.  This function always returns true.

## Syntax

	vungle.showCacheFiles()

## Examples

``````lua
-- outputs the list of files in the Vungle cache to the log
vungle.showCacheFiles()
``````
